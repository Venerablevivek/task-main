Step 1 - Download the Zip and extract it.

Step 2 - Open folder in VS code and open terminal

Step 3 - create a file .env.local on the root of the project and create these variables 
    MONGO_URL=YOUR_MONGODB_URL
    DB_NAME=YOUR_DB_NAME
    NEXT_PUBLIC_BASE_URL=http://localhost:3000
    CORS_ORIGINS=*
    JWT_SECRET=YOUR_JWT_SECRET

Step 4 - npm install
Step 5 - npm install cross-env
Step 6 - npm run dev
